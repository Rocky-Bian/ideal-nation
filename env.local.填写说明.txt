【重要】真正的配置文件名是：.env.local（前面有个点）

在 Cursor 里打开它的最快方法：
  按 Cmd + P ，输入：  .env.local  ，回车

文件完整路径：
  /Users/mac/Desktop/ideal-nation/.env.local

把下面几行填进 .env.local 后保存，然后运行 npm run dev

--- 复制以下内容到 .env.local ---

NEXT_PUBLIC_SUPABASE_URL=从 Supabase → Settings → API → Project URL
NEXT_PUBLIC_SUPABASE_ANON_KEY=从 Legacy 标签 → anon public
SUPABASE_SERVICE_ROLE_KEY=从 Legacy 标签 → service_role（secret）

DEEPSEEK_API_KEY=你的 DeepSeek API Key

CRON_SECRET=随便写一串本地密码，例如 my-secret-123

--- 结束 ---

Supabase SQL 还没跑的话：
  在 Supabase 网页 → SQL Editor → 依次粘贴运行
  1) supabase/schema.sql
  2) supabase/seed.sql
